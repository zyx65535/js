• Задача N 1

  • var a = 1, b = 1, c, d;

  • c = ++a; alert(c); // 2

    • Ответ - Префиксная форма инкремента. Инкремент происходит до вывода

  • d = b++; alert(d); // 1

    • Ответ - Постфиксная форма инкремента. Инкремент происходит после вывода

  • c = (2+ ++a); alert(c); // 5

  • d = (2+ b++); alert(d); // 4

  • alert(a); // 3

  • alert(b); // 3
